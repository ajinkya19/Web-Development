Projects

Dynamic Feature Matching for Partial Face Recognition

Porject Lead

Technologies

Deep Learning / FCN / MR-CNN / Sparse Representation Classification / Dynamic Feature Matching

Key skills

HTML5 / CSS / Python / MySql / React
Description

Partially recognize a face from selected database by matching facial features and iris using Neural Networks.
1. Use Fully Convulational networks to process and arbitrary size of an image which was not previously possible with CNN.
2. Dynamic Feature Representation to divide an image into sub-images of smaller size and extract sub-features from generated sub-images
3. Dynamic Feature Matching is used to match the sb-images features recieved from DFR, when matching is successfull outputs of similar probes are sorted out.
4. Multiscale Representation is then used to implement multiple levels of DFR's so any probe image doesn't miss.
5. The final image is thus obtained with 98% accuracy.
